1. YUC 与 AGE 已统一为一个 `DataRegistryProvider`。
2. AGE 进入主档案、搜索、详情、收藏、资源、写作和审查流程。
3. AGE 六类分页接口、详情接口和播放页接口已接入。

10. AGE 解析器已处理属性换序、非法 URL 和异常 URI 编码。

## 真实数据同步

先做小范围验证：

```bash
node scripts/sync-age.mjs \
  --categories=japan \
  --max-pages=3 \
  --concurrency=1 \
  --delay=500 \
  --output=public/data/age-latest.json
```

确认快照结构、来源页、失败清单和前台读取正常后，再执行：

```bash
npm run sync:age
```

详情和播放页数量很大，先设上限：

```bash
node scripts/sync-age.mjs \
  --all-categories \
  --details --max-details=100 \
  --play --max-play=200 \
  --concurrency=1 \
  --delay=500
```

不得在未限速、未确认上游允许且没有磁盘容量评估时直接运行无限制完整播放页同步。
